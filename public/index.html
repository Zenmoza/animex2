API Documentation

Endpoints

  • GET /home - Mendapatkan semua postingan anime

    Contoh Request: GET http://localhost:3000/home

    Response: Mengembalikan daftar semua postingan anime dalam format JSON.

  • GET /anime/:id - Mendapatkan anime berdasarkan ID

    Contoh Request: GET http://localhost:3000/anime/1

    Response: Mengembalikan data anime dengan ID yang diberikan atau pesan 'Anime not found' jika tidak ditemukan.

  • GET /genre - Mendapatkan daftar genre

    Contoh Request: GET http://localhost:3000/genre

    Response: Mengembalikan daftar genre unik dari semua postingan anime dalam format JSON.

  • GET /genre/:genre - Mendapatkan postingan berdasarkan genre

    Contoh Request: GET http://localhost:3000/genre/action

    Response: Mengembalikan daftar postingan anime yang sesuai dengan genre yang diberikan.

  • GET /search - Mencari anime berdasarkan judul

    Contoh Request: GET http://localhost:3000/search?title=Naruto

    Response: Mengembalikan daftar anime yang judulnya mengandung kata kunci yang diberikan.

  • POST /add_post - Menambahkan postingan baru

    Contoh Request:

    
POST http://localhost:3000/add_post
Content-Type: application/json

{
    "id": "4",
    "title": "My Hero Academia",
    "genre": "Action"
}

    Response: Mengembalikan postingan yang baru ditambahkan dalam format JSON.

  • PUT /update_post/:id - Mengupdate postingan berdasarkan ID

    Contoh Request:

    
PUT http://localhost:3000/update_post/1
Content-Type: application/json

{
    "title": "Naruto Shippuden"
}

    Response: Mengembalikan postingan yang telah diperbarui dalam format JSON.

  • DELETE /delete_post/:id - Menghapus postingan berdasarkan ID

    Contoh Request: DELETE http://localhost:3000/delete_post/1

    Response: Mengembalikan status 204 No Content jika berhasil menghapus postingan.

  • POST /generate_api - Menghasilkan API key

    Contoh Request: POST http://localhost:3000/generate_api

    Response: Mengembalikan API key baru dalam format JSON.

  • POST /reset-hits - Mereset hits berdasarkan API key

    Contoh Request:

    
POST http://localhost:3000/reset-hits
Content-Type: application/json

{
    "api_key": "your_api_key_here"
}

    Response: Mengembalikan status sukses jika hits berhasil direset.

  • GET /docs - Melihat dokumentasi API

    Contoh Request: GET http://localhost:3000/docs

    Response: Mengembalikan halaman dokumentasi API dalam format HTML.
